Puente GPS para celular o computadora

Esta pagina toma la geolocalizacion del dispositivo y la envia al geoportal para que aparezca en Agronomia > Seguimiento GPS y telemetria en tiempo real. Funciona con celular, laptop o cualquier equipo que entregue ubicacion desde el navegador.

Endpoint

El puente publica lecturas en:

/api/agronomy/gps/ingest

Luego el modulo agricola las lee desde /api/agronomy/gps/live.

Nota de red

Si vas a conectar otro equipo en la misma red, abre esta pagina usando la IP local del computador donde corre server.ps1, por ejemplo http://192.168.1.50:8765/gps-bridge.html.

Importante

En algunos celulares, la geolocalizacion del navegador solo funciona en HTTPS o localhost. Si tu navegador bloquea el GPS al usar una IP local, el endpoint de ingestión ya queda listo para que una app externa o un atajo HTTP envie el mismo JSON al geoportal.
